About me

Hello! My name is Vasiliy Yorkin and this is my personal website.

Currently I’m doing blockchain R&D with the Substrate framework and EVM/Solidity. I’m interested in blockchain security, DeFi, statically typed functional programming, compilers, type theory and formal verification.

In this blog I’m going to write about some of the things I do. Occasionally I tweet about those and other things on Twitter. Most of my open source work is available on GitHub.
